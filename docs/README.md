# Documentation of Label Studio

## How to use and deploy Hexo

### Installing Dependencies

```shell
npm install
```

### Starting Development server

```shell
npm run server
```

Starts a local server. By default, this is at http://localhost:4000/.

### Deploying Documentation

```shell
npm run publish
```
